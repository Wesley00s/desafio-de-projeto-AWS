# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 16/04/2024
Empresa: Abstergo Industries 
Responsável: Wesley Rodrigues

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Wesley. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- AWS Lambda
- Execução de código sem provisionamento de servidor
- Utilizado para otimizar o processamento de tarefas em lotes, reduzindo custos operacionais e aumentando a escalabilidade.

Etapa 2: 
- Amazon S3 (Simple Storage Service)
- Armazenamento de dados escalável e durável na nuvem
- Redução de custos de armazenamento, backup automatizado e compartilhamento seguro de dados.

Etapa 3: 
- Amazon EC2 (Elastic Compute Cloud)
- Hospedagem de aplicativos escaláveis
- Redução de custos operacionais ao hospedar aplicações e websites, pagando somente pelos recursos utilizados.



## Conclusão
A implementação de ferramentas na empresa *Abstergo Industries tem como esperado  reduzir os custos operacionais, aumentando a eficiência e a produtividade*, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
